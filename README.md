## ✨ Key Features

- 🚀 **Futuristic UI**: Premium dark-mode design with glassmorphism, glowing borders, and smooth Framer Motion animations.
- 📊 **GitHub Stats Integration**: Live display of GitHub contributions, top languages, and repository statistics.
- 📁 **Filterable Projects**: A dedicated projects section with categories like Hackathons, Personal, and Open Source.
- 🎓 **Verified Credentials**: Showcase of Sololearn and industry certifications with live verification links.
- 📧 **Functional Contact Form**: Direct-to-email communication powered by **EmailJS** with real-time status feedback.
- 📄 **Resume Integration**: Instant "View" and "Download" options for a professional B.Tech CSE resume.
- 📱 **Fully Responsive**: Meticulously optimized for mobile, tablet, and desktop viewing experiences.

## 🚀 Getting Started

### Prerequisites
- **Node.js** (v18 or higher)
- **npm** or **yarn**

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/TrikamDevasi/trikam-glow-folio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd trikam-glow-folio
   ```

3. **Install dependencies**
   ```bash
   npm install
   ```

4. **Start the development server**
   ```bash
   npm run dev
   ```

## 🔑 Environment Variables

To enable the contact form, create a `.env.local` file in the root directory and add your **EmailJS** credentials:

```env
VITE_EMAILJS_SERVICE_ID=your_service_id
VITE_EMAILJS_TEMPLATE_ID=your_template_id
VITE_EMAILJS_PUBLIC_KEY=your_public_key
```

This project is optimized for deployment on **Vercel** or **Netlify**.

1. Fork this repository.
2. Link your GitHub account to your hosting provider.
3. Add your Environment Variables in the provider's dashboard.
4. **Deploy!**
